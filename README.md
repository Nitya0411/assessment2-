# StayFinder - Hotel Search App

A responsive hotel search application built using React. The project fetches hotel data from a REST API and allows users to search, filter, sort, and view hotel details.

## Features

- View hotel listings
- Search hotels
- Filter by location and rating
- Sort by price
- Hotel details page
- Favorites (LocalStorage)
- Dark mode
- Responsive design

## Tech Stack

- React
- HTML
- CSS
- JavaScript
- React Router
- Fetch API

## Installation

```bash
npm install
npm run dev
```
